## Brand Design System

The overall aesthetic should feel like a luxury fashion editorial rather than a traditional retail website.

Keywords:
- Elegant
- Editorial
- Feminine
- Boutique
- Modern
- Minimal
- Calm
- Timeless

Avoid:
- Corporate
- Busy
- Template-looking
- Over-designed
- Cartoonish
- Trendy
- Harsh contrasts

## Color Usage

Warm Ivory (#FCF9F6)
Use as the primary page background.

Soft Blush (#F6EEEB)
Use for alternating section backgrounds to create subtle separation.

Pale Rose (#F1E1DE)
Use sparingly for decorative backgrounds, soft gradients, image overlays, and subtle accents.

White (#FFFFFF)
Use for elevated content such as cards, forms, dialogs, and featured containers.

Primary Rose (#CF8C93)
Brand accent color.
Use for:
- Primary buttons
- Links
- Active navigation
- Icons
- Small highlights
- Decorative accents
- Section labels

Primary Rose Hover (#BE747E)
Use only for hover and active interaction states.

Rich Charcoal (#1F1F1F)
Use for:
- Headlines
- Navigation
- Icons
- Important emphasis
- High-contrast text

Warm Gray (#5A5552)
Use for:
- Paragraphs
- Supporting text
- Descriptions
- Labels
- Footer copy

Divider (#DDD2CA)
Use for:
- Borders
- Dividers
- Input outlines
- Vertical separators

--------------------------------------------------

## Typography

Heading Font:
Cormorant Garamond

Body Font:
Inter

Use the serif font only for major headings and titles.

Use the sans-serif font for all navigation, body copy, buttons, forms, labels, and supporting text.

Maintain a strong visual hierarchy through typography rather than excessive color.

--------------------------------------------------

## Layout

Maximum content width:
1280px

Comfortable content width:
1180px

Generous white space throughout.

Never allow content to feel crowded.

When uncertain, increase spacing rather than reducing it.

--------------------------------------------------

## Components

Cards
- White background
- 20px border radius
- Soft shadow
- Very light border

Buttons
- Fully rounded
- Soft hover animation
- Subtle elevation on hover

Images
- Large
- Editorial
- Natural lighting
- Rounded corners

Icons
- Thin line icons
- Minimal styling

Animations
- Fade
- Fade up
- Small movement
- 500–700ms
- Never distracting

--------------------------------------------------

## Design Philosophy

Photography should be the primary visual focus.

Color should guide attention rather than dominate the design.

Typography should establish hierarchy.

White space should communicate luxury.

The overall experience should feel refined, approachable, and professionally designed.

Future pages should follow this design language regardless of their content.
