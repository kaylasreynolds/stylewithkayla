**Foundation:** Permanent Prompt Foundation v3.0  
**Purpose:** Validate prompt assembly before automated prompt generation.

---

# Purpose

This document validates the Style with Kayla Brand Photography System by manually assembling a representative set of production-ready prompts.

Each prompt build combines approved modules from the Permanent Prompt Foundation and Photo Module Library into a complete prompt to verify that:

- identity preservation remains exact
- module transitions read naturally
- prompts assemble correctly
- brand standards are consistently maintained
- generated imagery is commercially usable

This document intentionally contains only a small number of representative prompt builds.

Once these builds have been validated, future prompts should be assembled automatically from approved modules rather than manually documented.

# How to Use This Test Set

1. Select one approved identity reference photograph.
2. Assemble the prompt using the selected modules.
3. Do not modify the permanent Identity, Hair, Engagement Ring, Camera, Lighting, Composition, or Quality sections during testing.
4. Record generation-specific issues in the Generation Review Record.
5. Promote recurring issues into the Permanent Prompt Foundation only after they appear across multiple prompt builds.
6. Preserve every generated prompt exactly as submitted.
7. Evaluate every generated image using the approved Quality Standards before approving the asset.

# Final Assembled Prompt

## Master Setup

Edit the approved reference photograph into a premium commercial branding portrait for Style with Kayla.

The result should feel naturally confident, approachable, refined, and unmistakably real.

---

## Identity Preservation

Preserve the subject's exact:

- facial structure
- forehead
- cheeks
- jaw
- chin
- nose
- lips
- smile
- smile lines
- eye shape
- eyelid structure
- iris size
- eyebrows
- freckles
- pores
- natural skin texture
- under-eye structure
- facial volume
- authentic asymmetry
- authentic body proportions

Do not:

- beautify
- idealize
- reshape
- contour
- slim
- enlarge eyes
- narrow nose
- sharpen jaw
- enlarge lips
- remove freckles
- smooth skin
- create artificial symmetry
- alter body proportions

The generated portrait should appear nearly indistinguishable from the approved identity reference.

---

## Hair Preservation

Preserve the subject's naturally voluminous copper-red spiral curls exactly.

Maintain:

- authentic curl pattern
- spiral ringlets
- realistic density
- natural fullness
- authentic flyaways
- realistic texture
- hairline
- individual curl definition
- natural movement

Default hairstyle:

**Signature High Curly Bun**

Maintain:

- full bun volume
- visible curl texture
- natural silhouette
- soft face-framing tendrils

Hair-down remains an approved alternative only when selected by the scene.

Never:

- replace curls
- simplify curls
- straighten hair
- reduce volume
- flatten the bun
- crop the hairstyle

---

## Engagement Ring Preservation

The engagement ring is a permanent identifying feature.

Whenever the left hand is visible preserve exactly:

- polished white gold
- ornate twisted vine band
- vintage-inspired detailing
- radiant-cut center diamond
- exact proportions
- exact placement
- exact scale

Never redesign, simplify, resize, or omit the ring.
